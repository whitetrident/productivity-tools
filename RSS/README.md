## 两个rss生成的网站，解决某些没有直接提供rss订阅的源

### [rsshub](https://docs.rsshub.app/)

RSSHub 是一个轻量、易于扩展的 RSS 生成器，可以给任何奇奇怪怪的内容生成 RSS 订阅源

### [werss](https://werss.app/)

搜索公众号获取RSS
